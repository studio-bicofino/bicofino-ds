# On Field — Asset Naming Convention

## Rules

- Lowercase only
- No accents or special characters
- Hyphen-separated words
- Descriptive names
- Use category prefix when the folder holds mixed content

## Folder Structure

```
on-field/
  athletes/
    guilherme-kerchner/
      portraits/     — cutout and editorial portrait images
      campaigns/     — campaign photography
      brand/         — wordmarks, personal logos, identity assets
  icons/             — custom UI icons (SVG)
  badges/            — achievement and event badges (SVG)
  passports/         — country/nationality flags or stamps (SVG)
  venues/            — venue identity marks (SVG)
  tournaments/       — tournament logos (SVG)
  clubs/             — club crests and logos (SVG)
  sponsors/          — sponsor brand marks (SVG)
```
